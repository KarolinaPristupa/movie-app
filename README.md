
## Назначение

Приложение для отображения каталога фильмов

## API:

[TMDB](https://www.themoviedb.org/)

## Функционал

- Получение данных о фильмах с внешнего API;
- Отображение списка фильмов с наивысшим рейтингом с возможностью пагинации;
- Реализация формы поиска;
- Использование роутинга для разделения страниц приложения;
- Возможность добавления фильма в список избранных с сохранением их в LocalStorage;
- Возможность просмотра более детальной информации о фильме;
- Интерфейс для просмотра списка избранных и возможности удаления из списка;

### Дополнительный функционал

- При загрузке фильмов реализован Loader;
- Оптимизация дизайна под мобильные устройства (до 420px);
- Использован React context;
- Использован TypeScript для типизирования и уменьшения количества потенциальных багов;
- Использованы алиасы для импортирования файлов;
- Организацию файловой структуры react приложения;
- Деплой приложения на платформу GitHub Pages;
- Настройка конфигурации eslint, prettier, husky;

## Описание экранов

Главная страница отображает список популярных фильмов, получаемых с внешнего API. Пользователь может просматривать актуальные киноленты, оценивать их описание и рейтинг. Предусмотрена пагинация, позволяющая постепенно загружать фильмы и удобно навигировать по каталогу. Для улучшения пользовательского опыта реализованы визуальные индикаторы загрузки и обработка ошибок. При клике на карточку фильма выводится более подробная информация о нем.

Пользователь имеет возможность сохранять понравившиеся фильмы и быстро возвращаться к ним при последующих посещениях приложения. Данные об избранных фильмах сохраняются локально, что обеспечивает доступ к ним даже после перезагрузки страницы. Интерфейс позволяет удалять фильмы из списка избранного.

## Используемые технологии

- **_eslint_** - линтер для JavaScript кода;
- **_prettier_** - инструмент для автоформатирования кода;
- **_yarn_** - менеджер пакетов;
- **_react_** - JavaScript-библиотека для создания пользовательских интерфейсов;
- **_typescript_** - строго типизированный язык для уменьшения количества потенциальных багов;
- **_vite_** - инструмент сборки;
- **_SCSS_** - препроцессор, который служит для быстрого написания CSS стилей;
- **_react-router-dom_** - библиотека для навигации между разными частями веб-приложения;

## Доступ к API

Приложение использует TMDB API для получения данных. В ряде стран сервис частично недоступен. Если фильмы не загружаются, рекомендуем использовать VPN.
